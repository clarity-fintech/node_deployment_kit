# CLRTY-1 Node Kit

### Deploy Custom Nodes. Control Execution. Power the Network.

The CLRTY-1 Node Kit gives builders and operators control over infrastructure on the network: execution environments, capital routing logic, private/public node operation, and DePIN participation.

## What Is a CLRTY-1 Node?

A CLRTY-1 node is a programmable execution unit capable of processing transactions, running custom logic, interfacing with HELIX execution, coordinating capital flows, and enforcing compliance rules.

## Node Types

1. **Public Network Nodes (Mainnet)** — validation, transaction verification, state sync, network security.
2. **Private Execution Nodes** — private mempool handling, restricted access, custom execution logic.
3. **Strategy Nodes** — AI-driven strategies, allocation flows, autonomous capital operations.
4. **Compliance Nodes** — jurisdiction rules, KYC enforcement, audit tracking.
5. **DePIN Nodes** — physical compute, storage, bandwidth, and edge execution resources.

## Core Capabilities

- Custom execution logic
- HELIX integration
- AI signal execution
- Liquidity coordination
- Security and isolation
- Node economics and rewards
- Sandbox testing and simulation
