# Configuration

Copy `.env.example` from the CLRTY project when needed. Never commit real secrets.

| Variable | Purpose |
|---|---|
| `CLRTY_API_BASE` | API base for node register/heartbeat |
| `CLRTY_ROOT` | Root path for local JSON registries |
| `CLRTY_L1_RPC` | L1 RPC endpoint |
| `CALENDLY_EMBED_URL` | B2B scheduling for external onboarding |
