# Deployment

1. Register node identity.
2. Send heartbeat at the tier interval (60s free, 15s sovereign).
3. Validate L1 readiness and node operations scripts.
4. For mainnet validator participation, complete external ops: certificates, HSM keys, governance approvals.

This kit includes identity + heartbeat + operational assets. Full validator mesh joining remains governed by CLRTY-1 mainnet operations.
